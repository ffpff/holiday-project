# 旅行规划表设计提示词

你是一位优秀的平面设计师和前端开发工程师，具有丰富的旅行信息可视化经验，曾为众多知名旅游平台设计过清晰实用的旅行规划表。

现在需要为我创建一个A4纸张大小的旅行规划表，适合打印出来随身携带使用。请使用HTML、CSS和JavaScript代码实现以下要求：

## 基本要求

### 尺寸与基础结构
- 严格符合A4纸尺寸（210mm×297mm），比例为1:1.414
- 适合打印的设计，预留适当的打印边距（建议上下左右各10mm）
- 采用单页设计，所有重要信息必须在一页内完整呈现
- 信息分区清晰，使用网格布局确保整洁有序
- 打印友好的配色方案，避免过深的背景色和过小的字体

### 技术实现
- 使用打印友好的CSS设计
- **提供专用的保存为图片按钮**，使用html2canvas库将整个页面转换为高质量PNG图片并下载
- **集成高德地图导航功能**，关键地点（景点、酒店、车站等）可点击跳转到高德地图进行导航
- 使用高对比度的配色方案，确保打印后清晰可读
- 可选择性地添加虚线辅助剪裁线
- 使用Google Fonts或其他CDN加载适合的现代字体
- 引用Font Awesome提供图标支持
- **引入html2canvas库**：`<script src="https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js"></script>`

### 专业设计技巧
- 使用图标和颜色编码区分不同类型的活动（景点、餐饮、交通等）
- 为景点和活动设计简洁的时间轴或表格布局
- 使用简明的图示代替冗长文字描述
- 为重要信息添加视觉强调（如框线、加粗、不同颜色等）
- 在设计中融入城市地标元素作为装饰，增强辨识度
- **地点名称应设计为可点击的链接样式**，使用不同颜色和下划线标识
- **为可导航地点添加地图图标**，提升用户体验

### 高德地图导航集成
- **景点地址链接**：使用高德地图Schema格式 `amapuri://openFeature?featureName=OnLocationDest&lat=纬度&lon=经度&dev=0&style=2`
- **酒店地址链接**：同样使用高德地图坐标导航
- **交通枢纽链接**：火车站、机场等重要交通节点
- **餐厅推荐链接**：特色餐厅位置导航
- **为每个可导航地点添加data属性**存储经纬度坐标
- **点击事件处理**：检测是否为移动设备，PC端提示下载高德地图APP

### 保存图片功能
- **实现saveAsImage()函数**：使用html2canvas库捕获整个旅行规划表
- **图片质量设置**：scale: 2（高分辨率），format: 'png'，backgroundColor: 'white'
- **文件命名**：使用旅行目的地和日期命名，如"深圳到大理三日游_2025春季.png"
- **加载提示**：显示"正在生成图片..."的提示信息
- **错误处理**：捕获转换失败的情况并给出友好提示

## 设计风格

- **实用为主的旅行工具风格**：以清晰的信息呈现为首要目标
- **专业旅行指南风格**：参考Lonely Planet等专业旅游指南的排版和布局
- **信息图表风格**：将复杂行程转化为直观的图表和时间轴
- **简约现代设计**：干净的线条、充分的留白和清晰的层次结构
- **整洁的表格布局**：使用表格组织景点、活动和时间信息
- **地图元素整合**：在合适位置添加简化的路线或位置示意图
- **打印友好的灰度设计**：即使黑白打印也能保持良好的可读性和美观
- **交互式链接设计**：可点击地点使用醒目的链接样式，悬停效果明显

## 技术实现代码示例

### JavaScript功能实现
```javascript
// 保存为图片功能
function saveAsImage() {
    const loadingDiv = document.createElement('div');
    loadingDiv.innerHTML = '正在生成图片...';
    loadingDiv.style.cssText = 'position:fixed;top:50%;left:50%;transform:translate(-50%,-50%);background:rgba(0,0,0,0.8);color:white;padding:20px;border-radius:10px;z-index:9999;';
    document.body.appendChild(loadingDiv);
    
    html2canvas(document.querySelector('.travel-plan'), {
        scale: 2,
        format: 'png',
        backgroundColor: 'white',
        useCORS: true
    }).then(canvas => {
        const link = document.createElement('a');
        link.download = '深圳到大理三日游_2025春季.png';
        link.href = canvas.toDataURL();
        link.click();
        document.body.removeChild(loadingDiv);
    }).catch(err => {
        console.error('图片生成失败:', err);
        alert('图片生成失败，请重试');
        document.body.removeChild(loadingDiv);
    });
}

// 高德地图导航功能
function navigateToLocation(lat, lon, name) {
    const isMobile = /Android|webOS|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(navigator.userAgent);
    const amapUrl = `amapuri://openFeature?featureName=OnLocationDest&lat=${lat}&lon=${lon}&dev=0&style=2&name=${encodeURIComponent(name)}`;
    
    if (isMobile) {
        window.location.href = amapUrl;
    } else {
        const webUrl = `https://uri.amap.com/marker?position=${lon},${lat}&name=${encodeURIComponent(name)}&src=myapp&coordinate=gaode&callnative=1`;
        window.open(webUrl, '_blank');
    }
}
```
